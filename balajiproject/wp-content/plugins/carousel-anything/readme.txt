== Description ==
This plugin is a extension for WPBakery Page Builder (Formerly Visual Composer) which will allow user to display any content in slider.You can use it as a <strong>Image slider,Post Slider,Testimonial Slider,Product Slider</strong> and many more. It is very easy to use and no knowledge of coding required.Plugin is mobile-friendly and allows creating completely custom slideshows and css3 banner rotators with images, videos or HTML content in slides. It is 100% responsive and touch enabled on touch devices. It comes up with beautiful animation styles which add interactivity to the site and engage user in it.
Carousel also supports YouTube or Vimeo videos that allows you to create video galleries.
It comes up with built in following elements
<ul>
	<li><strong>Post Carousel</strong></li> 
	<li><strong>Video Carousel</strong></li> 
	<li><strong>Testimonial Carousel</strong></li> 
	<li><strong>Image Over Image Carousel</strong></li> 
	<li><strong>More to come....</strong></li>
</ul>
